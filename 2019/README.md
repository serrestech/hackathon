# Install

## Install Gulp
```
npm install gulp -g
```

## Clone Git
```
git clone https://github.com/serrestech/hackathon.git

cd hackathon
```

## Install Dependencies
```
npm install
```

## Run
```
gulp
```
### (Gulp opens url http://localhost:8000)
