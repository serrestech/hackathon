ΘΕΜΑ

  • Algorithms in practice

    Δείτε το φετινό θέμα.

    Το 5o Serres Hackathon έχει ως στόχο την δημιουργία εφαρμογών που χρησιμοποιούν και αναλύουν δεδομένα που αφορούν αλγορίθμους! Το θέμα του Hackathon θα βασίζεται σε υπαρκτό πρόβλημα και θα περιλαμβάνει πραγματικά δεδομένα τα οποία θα πρέπει να αξιοποιηθούν.

    Πιο συγκεκριμένα, στις ομάδες θα δοθεί ένα εννιαίο τεχνικό ζήτημα προς επίλυση όπου θα περιλαμβάνονται ή θα ζητούνται μεταξύ άλλων οι ακόλουθες τεχνολογίες:

    • Algorithm implementation
    • Pattern recognition
    • Functional programming
    • Execution speed and performance
    • Code optimization
    • User Interface (UI) design

    Για πρώτη φορά φέτος το θέμα θα δοθεί στις ομάδες πριν προσέλθουν στον χώρο του Hackathon. Αυτό γίνεται για να προετοιμαστούν οι Ομάδες καλύτερα και να εμφανιστούν στο Hackathon με περισσότερο διαθέσιμο χρόνο αλλά και ένα πιο οργανωμένο πλάνο. Συνάμα, από τις 19:00 της Παρασκευής μέχρι και την Κυριακή στις 13:10 συμπληρώνονται 42 ώρες, όσο περίπου είναι και τα χιλιόμετρα στον Κλασικό Μαραθώνιο!

    Σκοπός του Ηackathon είναι να έρθουν οι συμμετέχοντες σε επαφή με ρεαλιστικά προβλήματα, να δοκιμάσουν τις δυνατότητές τους στην υλοποίηση πραγματικών εφαρμογών, να μάθουν να συνεργάζονται με άλλα άτομα, να εκπαιδευτούν στην πράξη στην χρήση εργαλείων όπως το git, να είναι σε θέση να παρουσιάσουν σε σύντομο χρόνο ένα έργο μπροστά στο κοινό, να έρθουν σε επαφή με τις έννοιες του ανοιχτού λογισμικού και τέλος να διασκεδάσουν μαθαίνοντας αφού αυτό που τους ένώνει όλους είναι το πάθος για την τεχνολογία.
